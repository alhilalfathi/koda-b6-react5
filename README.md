# Survey Perokok Apps
aplikasi ini akan merupakan web app survey untuk mendapatkan data perokok dan rokok apa yang pernah dicoba. Dibuat menggunakan react dan styling menggunakan tailwind.

## cara menggunakan
1. download semua file di repo ini
2. buka terminal
3. ketik npm run dev
